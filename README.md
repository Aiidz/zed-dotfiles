# Zed Config Windows

Minimalist Zed editor configuration for Windows development.

## Core Features
* **Run File**: `Ctrl + R` runs Python, Java, or C++ using a custom batch script.
* **Open Recent**: `Ctrl + Alt + O` switches projects in the current window.
* **Environment**: Native Windows setup.

## File Descriptions
* **settings.json**: Editor preferences and UI.
* **keymap.json**: Custom keyboard bindings.
* **tasks.json**: Task configurations for Zed.
* **run.bat**: Logic for executing different file types.

## Installation
Place all files in:
`%APPDATA%\Zed`
